This is a <a href="https://github.com/baresip/baresip">baresip</a> based SIP User Agent application for Android.

Currently baresip+ supports SIP based voice and video calls, mobile voice calls, SIP based voice conference calls and text messages, voicemail Message Waiting Indication as well as blind and attended call transfers. 

In SIP calls audio can be coded with Opus, AMR, Codec2, G.729, G.722, G.722.1, or PCMU/PCMA codecs. Security is achieved via TLS or WSS SIP signaling transport and ZRTP or (DTLS) SRTP media encapsulation. Video can be coded with VP9, VP8, H.265, H.264, or AV1 codecs. Security is achieved via TLS or WSS signalling transport and ZRTP or (DTLS) SRTP media encapsulation.

Development of baresip+ is motivated by need for a secure, open source SIP based VoIP User Agent for Android that does not depend on proprietary, third party push notification services.  Mobile calling was added in order to avoid the need for two voice calling apps.

This application can be installed on Android devices running Android version 9 or later.  Mobile calling needs Android version 10 or later.

If you don't need video calling, you can instead of this app install its lighter weight sister app baresip.

Source code is available at <a href="https://github.com/juha-h/baresip-studio">GitHub</a>, where also issues can be reported.
